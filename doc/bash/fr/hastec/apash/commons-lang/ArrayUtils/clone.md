### ArrayUtils.clone

#### Example

```bash
ArrayUtils.clone  ""       ""               # failure
```

#### Arguments

* **$1** (ref(string[])): Name of the array to modify.
* **$2** (ref(string[])): Name of the array which will receive the clone

#### Exit codes

* **0**: When input arguments are arrays.
* **1**: Otherwise.

#### Output on stdout

* None

#### Output on stderr

* None.
