### ArrayUtils.isEmpty

#### Example

```bash
ArrayUtils.isEmpty ""              # false
ArrayUtils.isEmpty "myVar"         # false
```

#### Arguments

* **$1** (ref(string[])): The array to check.

#### Exit codes

* **0**: Whether the given array has no element.
* **1**: Otherwise.

#### Output on stdout

* None.

#### Output on stderr

* None.
