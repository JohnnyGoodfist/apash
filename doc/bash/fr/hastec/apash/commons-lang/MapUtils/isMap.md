### MapUtils.isMap

#### Example

```bash
MapUtils.isMap  ""          # false
MapUtils.isMap  "myVar"     # false
```

#### Arguments

* **$1** (string): Name of the hashmap.

#### Exit codes

* **0**: When the corresponding variable is a hashmap.
* **1**: Otherwise.

#### Output on stdout

* None.

#### Output on stderr

* None.
