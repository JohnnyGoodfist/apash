### StringUtils.reverse

#### Example

```bash
StringUtils.reverse             # ""
StringUtils.reverse ""          # ""
StringUtils.reverse "bat"       # "tab"
StringUtils.reverse "bat man"   # "nam tab"
```

#### Arguments

* **$1** (string): The string to reverse.

#### Exit codes

* **0**: When result is displayed.
* **1**: Otherwise.

#### Output on stdout

* The reversed string.

#### Output on stderr

* None.
