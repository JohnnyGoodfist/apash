Sorts the specified array into ascending natural order.

### Array.sort

#### Example

```bash
myArray=("a" "b" "c")
Array.sort "myArray"  # ("a" "b" "c")
```

#### Arguments

* **$1** (ref(string[])): The array to sort.

#### Exit codes

* **0**: True Whether the array is sorted according to natural ordering.
* **1**: Otherwise.

#### Output on stdout

* None.

#### Output on stderr

* None.
